## 🚀 Installation

To run this project locally, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/itxnargis/trendz-ecommerce-frontend.git
   cd trendz-ecommerce-frontend

   
2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**

    Create a .env file in the root directory and add your environment variables. You might need to set the REACT_APP_BACKEND_URL for the backend API.

   ```bash
    REACT_APP_BACKEND_URL=http://localhost:5000
   ```

4. **Run the application:**

   ```bash
   npm start
   ```

5. **Open the browser:**

   ```bash
   Navigate to http://localhost:3000 to see the application running.
   ```

## 🕹 Usage
